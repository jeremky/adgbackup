# adgbackup.sh

Sauvegarde simplifiée de l'application AdGuardHome.

## Utilisation

Editez le fichier `adgbackup.cfg` pour y indiquer les éléments suivants : 

- L'emplacement du répertoire de AdGuardHome

- La destination des sauvegardes

- Le user utilisé (pour accorder les bons droits après sauvegarde)

Une fois votre fichier correctement configuré :

```bash
./adgbackup.sh
```
